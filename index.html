Задача про рюкзак (динамічне програмування)

Алгоритм розв’язання задачі рюкзака

Задача рюкзака полягає в тому, щоб обрати певний набір предметів, максимізуючи їхню загальну вартість, при цьому загальна вага вибраних предметів не повинна перевищувати обмеження рюкзака.

Алгоритм динамічного програмування для розв’язання задачі рюкзака включає такі кроки:

  1. Створюємо таблицю dp[i][w], де i — це індекс предмета, а w — поточний ваговий ліміт.
  2. Ініціалізуємо базові значення: dp[0][w] = 0 для всіх w (тобто без жодного предмета максимальна вартість дорівнює 0).
  3. Для кожного предмета i і кожного значення ваги w виконуємо: 
    
dp[i][w] = max(dp[i-1][w], dp[i-1][w - weight[i]] + value[i])
  4. Якщо предмет не поміщається (тобто weight[i] > w), то: 
    
dp[i][w] = dp[i-1][w]
  5. Після заповнення таблиці результат знаходиться у dp[n][W], де n — кількість предметів, а W — максимальна допустима вага рюкзака.

Цей метод дозволяє ефективно визначити максимальну вартість предметів, які можна вмістити в рюкзак.
